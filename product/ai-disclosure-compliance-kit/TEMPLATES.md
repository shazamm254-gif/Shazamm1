# Templates & Checklists

Copy-paste tools to make disclosure a 30-second pre-publish step instead of a
judgment call. Fill in the brackets.

---

## 1. Per-video decision flowchart (run this before every upload)

```
START
  |
  v
Could a reasonable viewer mistake this for real, unaltered
footage/audio of a real person, place, or event?
  |
  +-- NO (obviously illustrated/stylized/CGI, or AI used only for
  |        script/subtitles/music/editing)
  |        --> No disclosure needed. Publish normally.
  |
  +-- YES / NOT SURE
       |
       v
     Does it show a real, identifiable person saying/doing
     something they didn't (incl. a cloned voice)?
       |-- YES --> DISCLOSE
       |
       +-- NO
            |
            v
          Is it a realistic scene/event presented as if it
          really happened, but didn't?
            |-- YES --> DISCLOSE
            |
            +-- NO
                 |
                 v
               Is it real footage altered in a way that changes
               what viewers would understand happened?
                 |-- YES --> DISCLOSE
                 |
                 +-- NO
                      |
                      v
                    Touches real people / real current events /
                    political or election topics / health or
                    financial claims?
                      |-- YES --> DISCLOSE (treat as non-negotiable)
                      |-- NO   --> Disclosure likely not required,
                                    but when unsure, disclose anyway.
```

## 2. Paste-ready disclosure text

**YouTube — video description, first line:**
```
This video contains AI-generated/altered content. [Brief note on what's
synthetic, e.g. "Visuals are AI-generated; narration is an AI voice."]
```

**YouTube — pinned comment (use alongside the in-app toggle, not instead of it):**
```
📌 Disclosure: this video uses AI-generated visuals/voice. The in-app
"Altered or synthetic content" label is enabled on this upload.
```

**TikTok — caption, appended after your hook:**
```
#AIGC AI-generated content — [one line: what's synthetic]
```
(Also enable the **AI-generated content** toggle in post settings — the
caption tag is a backup, not a substitute.)

**Instagram / Facebook — caption or first comment:**
```
This post includes AI-generated/altered media. [One line: what's synthetic.]
```
(Also enable the **AI info** label in post settings.)

**Generic / universal (use anywhere a platform lacks a native toggle):**
```
[Channel name] discloses: this content includes AI-generated or AI-altered
[visuals/voice/footage]. [One sentence on what specifically is synthetic.]
```

## 3. On-screen label template (for sensitive-topic videos)

For videos on health, news, elections, finance, or depicting real people —
add a small persistent on-screen label, even if the platform doesn't force
one, since these are the categories most likely to get extra scrutiny:

```
[Bottom-third, small, present for the whole video]
"AI-GENERATED / SYNTHETIC MEDIA"
```

## 4. Compliance audit log

Keep a running log (spreadsheet or this table) so you have a dated record of
your process if a platform ever flags a video.

| Date | Video / Post | Platform | Disclosure needed? | Toggle enabled? | Description text added? | Notes |
|---|---|---|---|---|---|---|
| [YYYY-MM-DD] | [title] | [YouTube/TikTok/IG] | [Y/N] | [Y/N] | [Y/N] | [anything unusual] |

Review this log monthly. If you're consistently marking "no disclosure
needed," spot-check a few against §1's flowchart to make sure you're not
drifting into "realistic" territory without noticing (AI video/voice tools
get more photorealistic every few months — recheck your assumptions
periodically).

---

## 5. Quick pre-publish checklist (print this)

- [ ] Ran the one-question test (§1) on this video
- [ ] If yes/unsure: disclosure toggle enabled on the platform
- [ ] Description/caption disclosure text pasted in
- [ ] On-screen label added if the topic is sensitive (health/news/
      elections/finance/real people)
- [ ] Logged in the audit log
- [ ] Separately confirmed this video also meets the platform's "original
      content" bar if you're relying on it for monetization
