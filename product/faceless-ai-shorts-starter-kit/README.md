# The Faceless AI Shorts Starter Kit

**Everything you need to launch a faceless, AI-generated YouTube Shorts channel —
and actually grow it.** No camera, no face, no on-screen presence. Just a
repeatable system for turning AI tools into a channel that compounds.

This is the exact system used to build speculative-science Shorts channels,
packaged so you can point it at *any* niche.

## Quick start (your first 7 days)

1. **Day 1 — Pick a niche.** `GUIDE.md` §1 walks you through choosing and
   validating one that's AI-friendly, monetizable, and has room to grow.
2. **Day 1 — Configure.** Copy the niche config template from `TEMPLATES.md`
   into `tools/niche.json` and fill it in for your niche. Every tool now works
   for you.
3. **Day 2 — Set up the pipeline.** `GUIDE.md` §3 lists the exact AI tools for
   visuals, voiceover, and editing, and how to batch.
4. **Day 3 — Generate ideas + scripts.** Run the idea generator, then write your
   first 10 using the script template. (See `docs/` for 10 finished examples.)
5. **Days 4–6 — Produce in one batch.** Generate visuals, record all voiceovers,
   edit, and schedule.
6. **Day 7 — Launch + commit.** Start posting daily. Run every upload through
   the metadata linter first.

The whole point: **build the system once, then feed it.** Consistency beats
perfection every single time on Shorts.

## The one rule that decides everything

A faceless channel doesn't fail because of the niche. It fails because the
creator stops posting. This kit is built to make posting *fast and repeatable*
so you don't stop. Treat the 30-day plan in `GUIDE.md` as non-negotiable.
