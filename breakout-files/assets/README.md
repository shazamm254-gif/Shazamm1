Optional assets, referenced from `config.yaml`:

- `music.mp3` — background bed. If present, it's looped, trimmed to the
  video's length, ducked (`video.music_duck_db`, default -18dB), and mixed
  under the narration. If absent, the video is narration-only.
- `stock/` — fallback frames used when FLUX image generation fails. See
  `stock/README.md`.
- `placeholders/` — auto-generated by `--dry-run` (git-ignored, safe to delete).
